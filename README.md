This repository holds the data files and preliminary analysis for a research proposal entitled "Premier League football as a generator of crime: a spatial analysis of stop and search data surrounding mass gatherings". This research proposal was conducted as part of the module "Wider Topics in Data Science" on the MSc Human and Social Data Science degree at the University of Sussex.

**Contained in this repository**:
1. football_stats (data folder):
   This folder contains 17 datasets which correspond to the 17 stable Premier League football teams in the 23/24 and 24/25 seasons. These      datasets contain statistical information about the home games for each team, including the date and time of each match, the score,          opponent, and whether that match would be considered a derby.

2. fooball-incident-statistics:
   This file has been pulled directly from the gov.uk, and contains information released each year regarding incidents that occur at           football matches. The particularly relevant information from this dataset has been used to create the datasets in football_stats.

3. stop_and_search_data:
   This folder contains over 150 datasets pulled from the data.police.uk repository. These correspond to the stop and search data from the     police constabularies surrounding the aformentioned stable Premier League football teams for each month of the 23/24 and 24/25 seasons.

4. analysis
   This file contains the python code and appropriate outputs from the preliminary analysis conducted for this research proposal. This         includes the processing of some of the data regarding Newcastle United, and outputs such as an interactive map.
